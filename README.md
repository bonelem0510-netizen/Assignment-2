# Assignment-2
I have made the repository for my assignment and will be adding further information as I work on it more, so with time
This app has been made to help users distinguish between Myths and Hacks by having them take a true or false quiz. The app is called Hack or Dud!
The user uses the application by pressing the "START" button to move onto the questions page.
From there, they press the "TRUE" button or the "FALSE" button based on the question that appears at the top.
They then press the "NEXT" button to move to the next question until they are done and taken to the score screen.
On the score screen, they can see what their score was, if they are a Hack or Dud, review the previous questions and answers, and exit the app.
